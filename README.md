# EDEN

**Early Distributed Environment Network**

EDEN is an open-source collaborative game/simulation project. The goal is to build an early-world environment through community contributions, one reviewed pull request at a time.

## Current Version

**v0.0.1**

Current features:

- Animated third-person player
- WASD movement
- Jump/falling animation
- Mouse camera control
- Smooth camera collision
- Test platform

## Contributing

Contributors should fork the repository, make changes in their own branch, and submit a pull request back to the official repository.

All changes must be reviewed before being merged into `main`.

## Project Status

This project is in very early prototype stage.
