# Asset Licenses

This file tracks the licensing and ownership status of assets included in The EDEN Project.

## General Rule

Do not contribute assets unless you created them yourself or have permission to use and distribute them as part of The EDEN Project.

Assets include, but are not limited to:

- 3D models
- Textures
- Animations
- Audio
- Music
- Images
- Icons
- Fonts
- Environment objects

## Contributor Requirement

Any pull request adding assets must state:

1. Who created the asset.
2. Where the asset came from.
3. What license applies.
4. Whether The EDEN Project has permission to use and distribute it.

Assets with unclear ownership may be rejected or removed.
