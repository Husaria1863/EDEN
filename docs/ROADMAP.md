\# EDEN Roadmap



This roadmap outlines the early development direction for The EDEN Project.



\## v0.0.1 — Prototype Foundation



Current status:



\- Third-person player

\- Player movement

\- Jump/falling animation

\- Idle and walk animations

\- Mouse camera control

\- Smooth camera collision

\- Test platform



\## v0.0.2 — Project Structure



Planned:



\- Clean folder structure

\- Contributor documentation

\- Pull request template

\- Asset licensing documentation

\- Basic feature proposal system



\## v0.0.3 — Environment Foundation



Planned:



\- Basic terrain

\- Simple environmental objects

\- Collision standards for environment assets

\- Lighting improvements

\- Placeholder sky/world settings



\## v0.0.4 — Interaction Foundation



Planned:



\- Basic object interaction

\- Pick-up system

\- Simple inventory prototype

\- Basic resource objects



\## v0.0.5 — Early Simulation Systems



Planned:



\- Day/night cycle

\- Temperature placeholder system

\- Simple hunger/thirst prototype

\- Basic survival loop



\## Long-Term Direction



The long-term goal is to build an open collaborative early-world simulation where contributors can add researched, reviewed, and tested systems through pull requests.
